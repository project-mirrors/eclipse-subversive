2. Information

2.1. System Reqirements

   Subversive requires JDK 1.4, JDK 1.5 or JDK 1.6 and supports following versions of Eclipse:
    * 3.2 family (tested with 3.2.0, 3.2.1)
    * 3.3M6 (tested with 3.3M6)
    
   Subversive supports Subversion versions 1.1, 1.2, 1.3 and 1.4 (with Native 1.4.3 and 
   SVNKit 1.1.1 clients only).

2.2 Known bugs

	* JavaSVN implementation of interactive merge features does incorrect merge at random time
