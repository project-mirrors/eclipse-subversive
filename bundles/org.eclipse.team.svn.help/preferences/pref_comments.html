Comments and Input History

These preferences can be changed on 'Team>SVN>Comments and Input History' properties page. On this page user can create, edit or remove comment templates for committing, preview selected template. These templates can be used later while performing sharing or commit operations. To add a comment template press 'New' and simply type it in an editable space of a dialog.

Tip:

This feature is very useful, when the user has, for example, some standardized comment type. Saving it to templates will essentially save his time and conformably quicken team work.

Option Description Default
Include 'tsvn:logtemplate' property value in commit log templates Specifies either include or not 'tsvn:logtemplate' property value in commit log templates. Enabled
Include user defined templates in commit log templates Specifies either include or not user defined templates in commit log templates. Enabled
