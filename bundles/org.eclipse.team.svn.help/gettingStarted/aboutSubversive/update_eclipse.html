Subversive update instructions (Eclipse.org version installed)

Subversive can be simply updated from within Eclipse using update manager. To update Subversive:

  1. Ensure you have enabled Subversive and Subversive Connectors update sites at 'Preferences > Install/Update > Available Software Sites'
  2. Select menu item 'Help > Check for Updates'



  3. Then mark software you want to update and follow through update wizard steps.
