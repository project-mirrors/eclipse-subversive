Hot Keys

To customize the key assist setting refer to General > Keys preferences page. The Subversive keys combination set follows.

Keys combination Description
CTRL+ALT+S Start synchronization
CTRL+ALT+C Start commit action
CTRL+ALT+U Start update action
CTRL+ALT+P Start patch creating action
CTRL+ALT+I Start adding to svn:ignore action
CTRL+ALT+K Start resource locking action
CTRL+ALT+W Start properties editing action
CTRL+ALT+T Show resource history
CTRL+ALT+A Show annotations
CTRL+ALT+L Compare with latest revision
CTRL+ALT+R Compare with specified revision
