Subversive update instructions

If the user has previously installed Subversive from eclipse.org update site

The user is able to update Subversive using Eclipse update manager and the same steps as for installation procedure. The only difference is skipping steps 3 and 4 (creating new update sites) and ignoring migrating projects and settings in step 11.

If the user has previously installed Subversive from polarion.org update site

The user is able to update Subversive using Eclipse update manager and the same steps as for installation procedure. But the user shouldn't uninstall his version of plug-in and specify the update sites offered in steps 3 and 4 (creating new update sites). Also the user should pay attention to the migrating projects and settings in step 11. The last is obvious thing to do if the user doesn't want to have his working copy crashed. Then and only after the described steps the user is able to safely uninstall the previous version of Subversive plug-in.
